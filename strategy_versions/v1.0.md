# v1.0 正式策略

- 状态：正式生效
- 配置来源：`config/strategy.yaml`
- 策略定位：热点质量 × 技术结构 × 风险过滤观察池
- 生效方式：人工确认

## 核心原则

- 热点只做第一层过滤，不等同于投资价值。
- 真实涨跌幅、收盘价、20日均线和60日均线覆盖达标后，才形成正式技术结构观察池。
- 数据不足时降级为候选观察池。
- 每日复盘只生成建议或 draft，不自动修改正式策略。
- 所有输出为研究辅助，不构成投资建议。

## 版本参数

参数以 `config/strategy.yaml` 为唯一运行时正式来源。本文件用于记录版本意图和人工确认状态。
